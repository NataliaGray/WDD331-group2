Theme: Campsite

The theme that we choose is a campsite at night. We want to create a big background image with main elements like the mountain, the sky, the trees, the campsite and maybe a lake. We want to have the page go down the mountain.

The mountain and the color scheme will be a uniting element. We want to have the dark sky with animated stars. We divided the content into 6-7 sections. We want to add something unique and possibly animated to each section. At the bottom of the page we want to have the campsite with either fire or a lake. We want the end to be warmer. We want to try and explore CSS and SASS and use the knowledge we get studying different modules. Even though the color scheme is dark we want to add elements to it to make it feel welcoming and calming, almost cozy. The fonts that we want to use should be well grounded, simple, and down to earth.

Here are the possible unique elements for different sections: twinkling stars, moving clouds, shining moon, fireflies in the forest, an owl with moving eyes, a bird picking from the side, water moving, text moving, fire moving, flashlight shining. Stars, moon and the clouds could be at the top. While other elements could be used in the lower sections. The fireflies can be closer to the bottom. We will decide when we finish the base background image.

Colors

We chose this color palette because we think it is really good for our theme. Russian Violet and Prussian Blue are for the mountains. We can use one of them for the mountain and the other one for the sky. We can make stars, moon, fireflies and other animated elements yellow or Lavender Web color. Adding warming elements could make the night not so cold and dark. Lavender web also could be a good color for the text. We decided to experiment and see. Y In Mn Blue is for the lake water (if we're going to add it). Green Sheen is great for the trees. We like the idea of having the bottom warm with fire and the camp to create a feeling of cosiness at the foot of the mountain. Another option that we're considering is having a lake at the bottom. water. Orange Yellow and Rust are great for creating the fire. Rust could also be used to the tree trunk if needed.
